//......................How to Run This Project on Your Computer........................//

1. Clone the repository.
   Open your terminal (PowerShell on Windows, terminal on macOS/Linux) and run:
   :> git clone https://github.com/nikodemo-bukwimba/Restaurant-management-system.git

2. Navigate to the cloned folder:
   :> cd Restaurant-management-system

3. Open the project in VS Code (optional but recommended):
   :> code .

4. Create and activate a virtual environment
   Windows (PowerShell):
   :> python -m venv venv
   :> .\venv\Scripts\Activate

   macOS/Linux:
   :> python3 -m venv venv
    :> source venv/bin/activate

5. Install project dependencies:
   :> pip install -r requirements.txt

6. Run the project:
   :> python manage.py runserver

7. Open the development server in a browser
   After running the above command, you will see:
   Starting development server at http://127.0.0.1:8000/

   Hold CTRL and click the link (Windows) or open it manually in your browser.
